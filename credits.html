Asset sources and licensing

TotTaps uses original visuals, procedural sounds, and carefully checked third-party assets only when the license allows safe use in the app.

Current assets

The current TotTaps app primarily uses original HTML, CSS, JavaScript, SVG, canvas visuals, procedural Web Audio sounds, and the TotTaps logo asset.

No third-party game art or sound packs are currently listed for attribution.

Future assets

When we add sprites, sound effects, music, or UI packs, each asset will be tracked with source URL, creator, license, download date, and required attribution.

Preferred licenses

Original TotTaps assets first, then CC0/public domain. CC-BY assets can be used only when attribution is shown here.

Avoided content

No copied characters, ripped sprites, trademarked game names, brand-like mascots, or unclear personal-use assets.

TotTaps asset policy lives in ASSET_POLICY.md.
