# TotTaps Assets

Place game artwork and sounds here only after checking the license.

Recommended subfolders:

- `original/` - artwork, sprites, UI, and sounds created for TotTaps.
- `cc0/` - public domain / CC0 assets.
- `credited/` - assets that require attribution.
- `audio/` - sound effects and music.

For each third-party asset, include a small `.txt` note beside it with source URL, author, license, and download date.

See `../ASSET_POLICY.md` before adding assets.

Also add the asset to `manifest.json`. If attribution is required, update `../credits.html` before release.
